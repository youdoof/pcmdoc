# executeCode

```pcm

```

## Description

Unofficial command to execute SmallTalk code through PCM scripting.

> Since Calypso X.Y

## Parameters

| Name       | Type   | Optional | Description               |

| ...        | Any    | Yes      | Remaining arguments.      |
